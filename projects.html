George-Droid

ИИ-компаньон для стримов, вдохновлённый Neuro-Sama.

ACTIVE PROJECT

Описание

George-Droid — многофункциональный стриминговый ИИ-ассистент на Python. Он слушает ваш голос, отвечает с юмором и взаимодействует с чатом Twitch как настоящий со-ведущий.

Работает через Together AI (с моделями типа Meta Llama 4 Scout Instruct или любой совместимой моделью Together AI), STT через Faster-Whisper, озвучка через Piper, и система контекстной памяти (RAG). Интеллект бота зависит от выбранной модели.

Возможности

  • Together AI LLM для умных ответов
  • Faster-Whisper STT + VAD
  • Piper TTS для озвучки
  • Twitch чат-бот
  • Монологи во время тишины
  • Контекстная память (RAG)

Технологии

Python Together AI Faster-Whisper Piper TTS Twitch API FAISS Sentence Transformers

NeuraBox

Программа для запуска локальных нейросетей с широким функционалом.

IN DEVELOPMENT

Описание

NeuraBox — это мощное и гибкое приложение-ИИ ассистент, разработанное для работы с различными нейронными сетями и обеспечивающее удобный пользовательский интерфейс для генерации текста, изображений и видео.

NeuraBox предназначен для энтузиастов ИИ, исследователей и разработчиков, которые ищут приватный и эффективный способ взаимодействия с локальными моделями ИИ.

Основные Возможности

  • Система чатов с вкладками
  • Поддержка множества моделей
  • Локальная обработка
  • Интерфейс с темной темой
  • Чат в реальном времени
  • Синхронизация задач

Технологии

Python Transformers PyQt/Tkinter Hugging Face PyTorch CUDA

FIREX BOT

Open-source Discord бот (уведомления, модерация, уровни, мини-игры).

LIVE & MAINTAINED

Описание

FIREX BOT — это многофункциональный Discord-бот с открытым исходным кодом, разработанный для упрощения управления сервером и добавления интерактивных элементов для сообщества.

Бот предоставляет набор инструментов, включая уведомления о начале стримов на Twitch, команды модерации, систему уровней на основе активности пользователей и развлекательные мини-игры.

Ключевые Функции

  • Уведомления Twitch
  • Модерация
  • Система Уровней
  • Мини-игры
  • Открытый Исходный Код
  • Локальное Развертывание

Технологии

Python Discord.py SQLite Twitch API Asyncio

My Time

V2 IN DEVELOPMENT
