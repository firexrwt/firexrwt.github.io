FIREX BOT

Beschreibung

FIREX BOT ist ein multifunktionaler Open-Source-Discord-Bot, der entwickelt wurde, um die Serververwaltung zu vereinfachen und interaktive Elemente für die Community hinzuzufügen. Ursprünglich für meinen persönlichen Server erstellt, wird der Bot nun für den globalen Einsatz vorbereitet.

Der Bot bietet eine Reihe von Werkzeugen, darunter Benachrichtigungen über den Start von Twitch-Streams, Moderationsbefehle, ein aktivitätsbasiertes Benutzerlevelsystem und unterhaltsame Minispiele.

Das Projekt ist Open-Source, was es anderen Benutzern ermöglicht, es einfach lokal bereitzustellen und an ihre Bedürfnisse anzupassen.

Hauptfunktionen

  • Twitch-Benachrichtigungen: Benachrichtigt den Server, wenn angegebene Streamer live gehen.
  • Moderation: Eine Reihe von Befehlen zur Aufrechterhaltung der Ordnung auf dem Server (z.B. kick, ban, mute - *Befehle bei Bedarf präzisieren*).
  • Level-System: Benutzer sammeln Erfahrung und steigen im Level auf, indem sie Nachrichten senden, was die Aktivität fördert.
  • Minispiele: Eingebaute einfache Spiele zur Unterhaltung der Servermitglieder (*Spiele bei Bedarf präzisieren*).
  • Open Source: Ermöglicht es jedem, den Code zu studieren, Beiträge zu leisten oder eine eigene Version des Bots bereitzustellen.
  • Lokale Bereitstellung: Konzipiert für die einfache Ausführung auf der eigenen Hardware des Benutzers.

Technologien

Die Haupttechnologien, die in FIREX BOT verwendet werden:

  • Hauptsprache & Bibliotheken:
    • Python 3.x
    • Nextcord (Asynchrone Bibliothek für die Discord-API)
    • Requests (Zur Interaktion mit externen APIs, z.B. Twitch)
    • Pillow & Colorthief (Bildverarbeitung und Farbextraktion)
    • Waitress (WSGI-Server - möglicherweise für Webhooks oder eine einfache API?)
    • Humanfriendly (Formatierungshilfsmittel)
  • Werkzeuge:
    • Git & GitHub
