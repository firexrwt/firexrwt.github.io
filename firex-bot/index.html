FIREX BOT

Описание

FIREX BOT — это многофункциональный Discord-бот с открытым исходным кодом, разработанный для упрощения управления сервером и добавления интерактивных элементов для сообщества. Изначально созданный для моего личного сервера, бот сейчас готовится к глобальному развертыванию.

Бот предоставляет набор инструментов, включая уведомления о начале стримов на Twitch, команды модерации, систему уровней на основе активности пользователей и развлекательные мини-игры.

Проект является open-source, что позволяет другим пользователям легко развернуть его локально и адаптировать под свои нужды.

Ключевые Функции

  • Уведомления Twitch: Оповещает сервер о начале трансляций указанных стримеров.
  • Модерация: Набор команд для поддержания порядка на сервере (например, kick, ban, mute).
  • Система Уровней: Пользователи получают опыт и уровни за отправку сообщений, стимулируя активность.
  • Мини-игры: Встроенные простые игры для развлечения участников сервера.
  • Открытый Исходный Код: Возможность для любого желающего изучить код, внести свой вклад или развернуть свою версию бота.
  • Локальное Развертывание: Бот спроектирован для легкого запуска на собственном оборудовании пользователя.

Технологии

Основные технологии, использованные в FIREX BOT:

  • Основной Язык и Библиотеки:
    • Python 3.x
    • Nextcord (Асинхронная библиотека для Discord API)
    • Requests (Для взаимодействия с внешними API, например, Twitch)
    • Pillow & Colorthief (Обработка изображений и извлечение цветов)
    • Waitress (WSGI сервер)
    • Humanfriendly (Утилиты для форматирования)
  • Инструменты:
    • Git & GitHub

Список Команд

Список всех команд в алфавитном порядке.

/add_bad_word
Добавляет плохое слово(а) в базу данных.
/add_streamer
Добавляет стримера в базу данных для отслеживания.
/bad_words
Показывает список всех плохих слов.
/ban
Банит участника сервера.
/clear_all_warns
Удаляет все предупреждения на сервере.
/clear_warns
Удаляет все предупреждения указанного пользователя.
/delete_message
Удаляет определенное сообщение по ID в выбранном канале.
/kick
Кикает пользователя с сервера.
/leaderboard
Показывает таблицу лидеров по уровню и количеству сообщений.
/messages_count
Показывает количество сообщений пользователя в канале.
/mute
Запрещает пользователю писать на сервере на некоторое время.
/profile
Показывает ваш уровень и количество написанных сообщений.
/remove_bad_word
Удаляет плохое слово из базы данных.
/remove_streamer
Удаляет стримера из базы данных отслеживания.
/rps
Играет с вами в Камень-Ножницы-Бумага.
/unban
Разбанивает пользователя по ID.
/unmute
Возвращает пользователю возможность писать в чат.
/warn
Выдаёт предупреждение пользователю.
/warns
Показывает список предупреждений пользователя.
